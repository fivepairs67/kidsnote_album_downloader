# 개인정보처리방침 (Privacy Policy)

최종 업데이트: 2026-03-03

본 개인정보처리방침은 Chrome 확장 프로그램 **Kidsnote Album Downloader**(이하 "확장 프로그램")의 데이터 처리 방식에 대해 설명합니다.

## 1. 처리하는 데이터

확장 프로그램은 사용자가 키즈노트 웹사이트에서 다운로드를 실행할 때 다음 데이터를 처리할 수 있습니다.

- 웹사이트 콘텐츠: 앨범/알림장 본문, 사진, 동영상, 첨부파일
- 개인 식별 가능 정보: 키즈노트 콘텐츠에 포함된 이름/표시명/식별 정보
- 개인 커뮤니케이션 성격 정보: 알림장 본문 등

위 데이터는 확장 프로그램의 기능 수행(파일 다운로드 및 폴더 정리)을 위해서만 처리됩니다.

## 2. 수집 및 저장 방식

- 모든 처리는 사용자의 브라우저(로컬 환경)에서 수행됩니다.
- 확장 프로그램은 다운로드 진행 상태 등 최소한의 설정/상태값을 Chrome Storage(로컬/세션)에 저장할 수 있습니다.
- 다운로드된 파일은 사용자의 기본 다운로드 폴더(또는 브라우저 다운로드 설정 경로)에 저장됩니다.

## 3. 외부 전송 및 제3자 제공

- 개발자 서버로 사용자 데이터(콘텐츠, 로그인 정보, 개인정보)를 전송하지 않습니다.
- 사용자 데이터를 판매하거나 광고/프로파일링 목적으로 사용하지 않습니다.
- 확장 프로그램은 기능 수행을 위해 키즈노트 도메인 및 미디어 CDN 도메인에만 접근합니다.

## 4. 권한 사용 목적

- `tabs`: 현재 활성 탭의 키즈노트 페이지 확인 및 연결
- `downloads`: 파일 다운로드 실행
- `storage`: 상태/로그/설정의 로컬 저장
- `host permissions (kidsnote.com, kakaocdn.net)`: 키즈노트 콘텐츠 조회 및 미디어 다운로드

## 5. 데이터 보관

- 브라우저 세션 데이터는 브라우저 종료 시 초기화될 수 있습니다.
- 로컬 저장 데이터와 다운로드 파일의 삭제/관리는 사용자가 직접 수행할 수 있습니다.

## 6. 아동 데이터 관련 안내

본 확장 프로그램은 보호자/사용자가 본인 계정으로 접근 가능한 키즈노트 콘텐츠를 개인 백업 목적으로 저장하는 도구입니다.  
사용자는 관련 법령 및 서비스 이용약관을 준수하여 사용해야 합니다.
