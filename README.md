# 👶 Kidsnote Downloader (앨범/알림장 다운로더)

[![Chrome Extension](https://img.shields.io/badge/Chrome-Extension-4285F4?style=flat-square&logo=Google-Chrome&logoColor=white)](#) [![License](https://img.shields.io/badge/License-MIT-green.svg?style=flat-square)](./LICENSE)

키즈노트(Kidsnote) 웹사이트의 **앨범**과 **알림장**을 기간별로 일괄 다운로드하는 Chrome 확장 프로그램입니다.

## ✨ 주요 기능

- 🔍 **모드 분리**: 앨범(`/service/album`)과 알림장(`/service/report`)을 각각 준비/다운로드할 수 있습니다.
- 🚀 **백그라운드 다운로드**: 화면 이동 없이 내부 API + Chrome Downloads API로 순차 다운로드합니다.
- 💎 **고화질 우선 선택**: 사진은 `original` 계열, 동영상은 `high` 계열 URL을 우선 사용합니다.
- 📁 **폴더 자동 정리**
  - 앨범: `YYYY-MM-DD-앨범제목/`
  - 알림장: `YYYY-MM-DD-반/작성자-id/` (동일 날짜/제목 충돌 방지)
- 📑 **본문/첨부 저장**
  - `text.txt`
  - `photos/*`, `videos/*`
  - 알림장 첨부파일: `files/*`
- 🛑 **중복 실행 방지**: 다운로드 중 재클릭해도 프로세스를 중복 실행하지 않습니다.

## 🛠 설치 방법

현재 크롬 웹 스토어에 정식 등록되어 있지 않으므로 **개발자 모드**를 통해 설치해야 합니다.

1. 크롬 브라우저를 열고 주소창에 `chrome://extensions` 를 입력하여 접속합니다.
2. 우측 상단에 있는 **개발자 모드** 토글을 켭니다.
3. 좌측 상단에 표시되는 **압축해제된 확장 프로그램을 로드합니다.** 버튼을 클릭합니다.
4. 이 깃허브 저장소(또는 소스코드)를 다운로드 받은 후 압축을 푼 폴더(`kidsnote_album_downloader/`)를 선택합니다.
5. 브라우저 우측 상단 주소창 옆의 **퍼즐 모양 아이콘(확장 프로그램)**을 클릭한 뒤, **Kidsnote Downloader** 옆의 **압정(고정) 아이콘 📌**을 눌러 도구 모음에 고정합니다.

## 📖 사용 방법

1. PC 크롬 브라우저에서 [키즈노트 웹사이트](https://www.kidsnote.com/) 에 로그인 합니다.
2. 크롬 우측 상단에서 확장 프로그램 아이콘을 클릭합니다.
3. 원하는 모드의 **준비/연결** 버튼을 클릭합니다.
   - `준비/연결(앨범)` 또는 `준비/연결(알림장)`
4. 필요 시 기간 필터(`from/to`, `YYYY-MM`)를 입력합니다. 비워두면 전체 기간입니다.
5. 원하는 모드의 **다운로드** 버튼을 클릭합니다.
   - `다운로드(앨범)` 또는 `다운로드(알림장)`
6. 진행 중에는 **정지** 버튼으로 안전 중단할 수 있습니다(현재 파일 처리 후 중단).

> 💡 안내  
> - 준비/연결 버튼을 누르면 모드별 기본 루트명이 자동 입력됩니다.  
>   - 앨범: `Kidsnote_album`  
>   - 알림장: `Kidsnote_report`  
> - 다운로드 파일은 기본 다운로드 폴더 아래 해당 루트에 저장됩니다.

## 📂 저장 예시

```text
Downloads/
  Kidsnote_album/
    2025-10-12-가을소풍/
      text.txt
      photos/001.jpg
      videos/001.mp4

  Kidsnote_report/
    2025-10-12-튼튼반-123456789/
      text.txt
      photos/001.jpg
      files/가정통신문.pdf
```

## 🔒 보안 및 개인정보

- **로컬 내장 처리:** 모든 다운로드 프로세스는 사용자의 PC 내 크롬 브라우저에서 직접 실행됩니다.
- **정보 전송 및 수집 없음:** 로그인 정보나 개인정보를 절대로 수집하거나 외부 서버로 전송하지 않습니다.
- **안전한 통신:** 이 확장 프로그램은 오직 `kidsnote.com` 도메인과 이미지/동영상이 실제 호스팅되는 카카오 CDN(`kakaocdn.net`)과만 통신합니다.
